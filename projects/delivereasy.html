Deliver’easy

Description : Plateforme de gestion de livraisons pour entreprises. Interface claire pour suivre les livreurs, clients, et tournées.

Fonctionnalités :

  • Création et gestion de livraisons
  • Suivi de statut (en cours, livrée, annulée)
  • Interface admin avec filtres et recherche

Technos : Vue.js, Node.js, PostgreSQL
